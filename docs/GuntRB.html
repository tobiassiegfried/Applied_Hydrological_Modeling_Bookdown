3.2 Gunt River Basin

3.2.1 Overview

3.2.2 Key Basin Characteristics

For the analysis of the key hydro-climatological characteristics, we first load the available decadal and monthly station data3. The data used in this Chapter can be accessed on the public GitHub repository Applied Modeling of Hydrological Systems in Central Asia and the ./data/AmuDarya/Gunt/ folder in there.

3.2.2.1 Hydrology

First, we load the available data into R.

# Load data records
fPath = fPath <- './data/AmuDarya/Gunt'
fName = '/StationData/gunt_data_cleaned.Rds'
data <- read_rds(paste0(fPath,fName))

Most of the data are only available at monthly resolution.

As compared to the Chirchik river basin, we only have one long-term operational gauging station in the Gunt river basin. The Station code is 17050. Its discharge data can be accessed and extracted from the Gunt dataset in the following way

q_17050_mon <- data %>% filter(type == "Q" & code == '17050' & resolution == 'mon')

When we plot the data, we see that we have a near complete monthly record from 1940 onwards. The data gap in the 1990ies was during the Tajik civil war which was a very difficult time for Tajikistan.

q_17050_mon %>% plot_time_series(date,

                                 .smooth      = TRUE,
                                 .interactive = TRUE,
                                 .title       = "",    
                                 .x_lab       = 'Year',
